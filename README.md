# DopaQiX

Candy-charged QiX-style beta, built to run with zero setup.

![DopaQiX front page](./docs/frontpage.png)

## Play

Extract [`CodexQiX-beta-static-v15.zip`](./CodexQiX-beta-static-v15.zip) and double-click `index.html`.

You can also open the project-root [`index.html`](./index.html) directly.

## Current Beta

- Static, zip-ready build
- `ship/` mirrors the latest playable files
- Includes cheat mode, music toggle, volume, speed, pause, and pickup tuning

## Notes

- Root static files are the current source of truth
- Older Vite/Pixi experiment files are still in the repo, but not required to play the beta
